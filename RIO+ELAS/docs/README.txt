# Publicação do site no GitHub Pages

1. Mova o conteúdo da pasta `docs` para a raiz do seu repositório ou mantenha a pasta `docs`.
2. Faça commit e push para o GitHub.
3. No GitHub, vá em "Settings" > "Pages".
4. Em "Source", selecione a branch `main` e a pasta `/docs`.
5. O GitHub Pages vai gerar um link do tipo:
   
   https://SystemHub00.github.io/FGM/
   
   ou
   https://SystemHub00.github.io/FGM/docs/

6. O site estará disponível nesse link, acessível de qualquer lugar.

Lembre-se: o formulário só enviará dados para a planilha quando acessado da sua máquina, pois o backend está rodando localmente.
