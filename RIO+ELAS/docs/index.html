Como fazer o formulário funcionar com Python

  1. Abra o terminal na pasta RIO+ELAS/backend-python.
  2. Instale as dependências:
    pip install -r requirements.txt
  3. Certifique-se de que o arquivo plated-field-474017-b8-e00d977b2612.json está na pasta RIO+ELAS (um nível acima do backend-python).
  4. Inicie o servidor:
    python app.py
  5. O backend ficará disponível em http://localhost:3001.
  6. O formulário deste site já está configurado para enviar para esse backend.
  7. Ao acessar este arquivo nesta máquina e enviar o formulário, os dados irão para a planilha Google Sheets.
Observação: O backend só funcionará localmente, ou seja, apenas nesta máquina.
Se quiser publicar o site, use o arquivo docs/index.html para o GitHub Pages.
